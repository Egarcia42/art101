Task 1: Making Guacamole

  1. Cut open avocados
  2. Remove pit and take out flesh
  3. Mash flesh in a bowl
  4. Cut open a lemon
  5. Squeeze lemon juice on mash in bowl and mix in
  6. Dice up a Tomato
  7. Add tomato to mash in bowl
  8. (Optional) Cut up Cilantro and add to mash in bowl

Task 2: Psuedocoding the game Snake

  1. Create arena border
  2. Spawn the food pixel --> this becomes the objective
  3. Spawn snake entity (1 pixel/grid square big)
  4. When up, down, right or left key pressed, move the snake entity in the indicated direction
  5. Check if snake entity is touching arena border; if so, stop
  6. Check if snake entity is touching its own body; if so, stop
  7. Check if snake entity is touching food pixel; if so:
  8. Add an extra pixel on the end of the snake entity
  9. Make the food pixel disappear
  10. Spawn a new food pixel in a random location
